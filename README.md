# Land2Build

This is the frontend for the Land2Build application, an AI-powered construction planning tool.

## Running the application

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
